## 📘 About react-native-swiftui-dsl

react-native-swiftui-dsl is a tool that helps you create user interfaces for React Native apps. It uses a style inspired by SwiftUI, which means you can build UI screens step by step by stacking simple instructions. This method is easy to follow and helps keep views neat and manageable.

This tool includes features like:

- **Chainable modifiers:** Change UI elements by connecting simple style commands.
- **Theme system:** Use preset colors and styles for consistent app looks.
- **Two-way bindings:** Link data and UI, so changes in one update the other automatically.

You don’t need to be a developer to try this out. If you want to explore how apps can be built with React Native using a SwiftUI-like method, this is a good place to start.

## 🛠 Understanding Key Features

### Chainable Modifiers

Instead of writing long code, you apply modifiers one by one.

Example:  
Start with a button, then add a background color, then rounded corners, then padding. Each step modifies how the button looks.

### Theme System

Pick colors and fonts from themes to keep the app style uniform. The theme sets the main colors, fonts, and sizes.

### Two-Way Bindings

Changes in data update the UI automatically. Likewise, changing the UI updates the data. This keeps your app interactive without extra setup.
